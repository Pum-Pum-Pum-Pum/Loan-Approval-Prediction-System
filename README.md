# Loan Approval Prediction System

Production-style ML project structure for loan approval prediction.

## Structure
- `data/raw/`: raw input datasets
- `notebooks/`: exploratory notebooks
- `src/`: source code for preprocessing, features, models, and utilities
- `artifacts/`: saved model artifacts
- `reports/`: generated reports and figures
- `configs/`: project configuration files

## Quick start
1. Review EDA in `notebooks/loan-approval-prediction-system.ipynb`
2. Run baseline pipeline with `python src/models/train_baseline.py`
